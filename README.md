# 🦈 SharkFit Mobile 🦈

> Train smarter. Eat better. Evolve every day.

---

🚧 **Projeto em desenvolvimento**

SharkFit é um aplicativo mobile desenvolvido em **React Native + Expo** para ajudar usuários a manter uma rotina saudável através do gerenciamento de treinos, alimentação, receitas e acompanhamento da evolução física.

Em breve este README será atualizado com demonstrações, capturas de tela e documentação completa.

---

## 🚀 Tecnologias

- React Native
- Expo
- TypeScript
- SQLite
- AsyncStorage
