# 📱 Sobre

SharkFit é um aplicativo desenvolvido em React Native com foco em ajudar pessoas a criarem uma rotina saudável através do gerenciamento de treinos, alimentação, receitas e acompanhamento da evolução física.

---

# ✨ Funcionalidades

- 🏋️ Cronograma de treinos
- 🍽️ Planejamento alimentar
- 📖 Cadastro de receitas
- 🔔 Lembretes inteligentes
- 📈 Acompanhamento da evolução
- 💾 Funcionamento Offline
- 📱 Interface moderna

---

# 🚀 Tecnologias

- React Native
- Expo
- TypeScript
- SQLite
- AsyncStorage
